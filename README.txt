Game Of Life MultithreadedHOW TO RUN FROM COMMAND LINEThis program runs off of command line args.To open it you have use cd command to get to your file repo.After that run the command java -jar L5.jar -filename- number of threads-filename is a .txt file which looks like this15 - Number of rows15 - Number of columns3 - Amount of turns10 - Number of alive cells12 313 212 012 112 2 ----- cordinates of every alive cell13 111 012 5 13 413 3 taken from example.txt in repo
